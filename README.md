"# VolumeWatcherWPF" 
ウィンドウズのボリューム状態を監視
再生デバイス
ボリューム
Mute状態
キーボードショートカットでボリューム調整も可能

C#-WPFで作成。

## 使い方
タスクトレイに常駐。
アイコンをクリックするとオプション画面が表示されるのでそこで設定してね。

## キーボードショートカット
[Alt]+[>]  Volume+
[Alt]+[<]  Volume-
[Alt]+[M]  Mute On/Off
